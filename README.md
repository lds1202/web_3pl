# 3PL 물류대행 플랫폼

화주사와 물류 창고업체를 연결하는 물류대행 매칭 플랫폼 웹 애플리케이션입니다.

> **최신 업데이트**: 2024년 10월 17일 - 회원가입 모달 개선, 헤더 네비게이션 최적화, 로그인 체크 기능 추가

## 🚀 주요 기능

### 1. 메인 페이지
- **클릭 가능한 로고**: 3PL Platform 로고 클릭 시 홈으로 이동
- **직관적 네비게이션**: 홈 버튼 제거로 깔끔한 메뉴 구성
- **두 개의 메인 섹션**: 창고 찾기, 고객사 찾기
- **프리미엄 창고 광고 영역**: 3개 특별 표시
- **최신순 정렬된 일반 리스트**: 창고 및 고객사 정보
- **로그인 체크 기능**: "자세히 보기" 클릭 시 로그인 상태 확인

### 2. 창고 찾기
- **고급 필터링**: 지역, 면적, 팔레트 수, 상품 유형, 보관 방식
- **아코디언 필터**: 접힌 상태로 시작, 부드러운 애니메이션
- **라디오 버튼 필터**: 면적과 팔레트 수를 5단계로 선택
- **프리미엄 창고**: 상단 광고 영역에 특별 표시
- **일반 창고**: 최신순 정렬, 페이지네이션
- **반응형 디자인**: 모바일에서 필터 토글

### 3. 고객사 찾기
- **검색 필터**: 지역, 면적, 상품 유형
- **일반 고객사 리스트**: 최신순 정렬
- **페이지네이션**: 효율적인 데이터 로딩

### 4. 등록 시스템
- **회원가입 모달**: 창고업체/고객사 선택 후 해당 등록 페이지로 이동
- **창고 등록**: 상세한 창고 정보 입력 (사업자등록번호, 팔레트 수 등)
- **고객사 등록**: 물류 요구사항 입력 (팔레트 수, 월평균 출고량 등)
- **면적 변환**: ㎡와 평 자동 환산 표시
- **실시간 유효성 검사**: 면적 관계 검증, 필수 필드 체크
- **ESC 키 지원**: 모달 닫기, 배경 클릭으로 닫기

### 5. 연락처 열람 시스템
- **결제 옵션**: 1회(50원), 10회(3,000원), 무제한(9,900원/월)
- **로그인 연동**: 사용자 인증 시스템
- **모달 기반**: 깔끔한 사용자 경험

### 6. 관리자 시스템
- **관리자 로그인**: ID: `admin`, PW: `1231`
- **URL 접근**: `/admin/login`, `/admin/dashboard`
- **승인 관리**: 검토 중인 창고/고객사 승인/거부
- **상세 보기**: 등록된 모든 정보 확인 (비밀번호 제외)

## 🚀 설치 및 실행

### 1. 프로젝트 클론
```bash
git clone [repository-url]
cd 1_3pl
```

### 2. 의존성 설치
```bash
npm install
```

### 3. 개발 서버 실행
```bash
npm run dev
```

```bash
npm run build
```

```bash
npm run preview
```

## 🛠 트러블슈팅

### 자주 발생하는 문제

#### 1. 포트 충돌
```bash
# 문제: Port 3000 is in use
# 해결: Vite가 자동으로 3001로 변경하거나
npm run dev -- --port 3002
```

#### 2. 의존성 설치 실패
```bash
# 문제: npm install 실패
# 해결: 캐시 클리어 후 재설치
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

#### 3. Hot Reload 문제
```bash
# 문제: 코드 변경이 반영되지 않음
# 해결: 개발 서버 재시작
Ctrl + C
npm run dev
```

#### 4. 모바일에서 필터가 보이지 않음
- **원인**: `isMobile` 상태가 제대로 감지되지 않음
- **해결**: `FilterSidebar` 컴포넌트 내부에서 `useEffect`로 감지

#### 5. 면적 변환이 작동하지 않음
- **원인**: `areaConverter.js` 유틸리티 함수 import 누락
- **해결**: `import { formatArea } from '../utils/areaConverter'` 추가
